Question

How do I map a network drive in Windows? 

Answer

For Windows 10

  1. Open File Explorer from the Start Menu (Start/Windows System/File Explorer) or by simply pressing the Windows Key and the letter E on your keyboard simultaneously.  File Explorer is often times pinned to your task bar as well and looks like a manilla folder.  You can also access it this way by clicking on that icon in the task bar.
  2. On the left hand side of the screen, look for the icon called This PC (or Windows) and click on it.
  3. At the top of the screen, in the menu bar, click the word Computer.
  4. A series of icons will appear below the menu bar.  Click Map network drive.
  5. This will bring up a window called Map Network Drive.  From here you can enter in the applicable information for the drive you intend to map
    1. First, specify a drive letter by clicking the drop down menu for Drive.
    2. Next, specify the specific folder you want to map.  For example: \\corpfs.nd.edu\sharedfoldername\
    3. Check the box for Reconnect at sign-in
    4. Uncheck the box for Connect using different credentials.  In most cases, this will force Windows to connect to the specified drive using your NetID/password.  If you need to connect to the drive using an alternate user name/password, you can check this box.
    5. Click Finish.
  6. Your new drive will now be mapped and mounted in the left pane of File Explorer.

For Windows 11

  1. Open File Explorer from the Start Menu (Start/Windows System/File Explorer) or by simply pressing the Windows Key and the letter E on your keyboard simultaneously.  File Explorer is often times pinned to your task bar as well and looks like a manilla folder.  You can also access it this way by clicking on that icon in the task bar.
  2. Select This PC from the left pane.  
  3. Click the three dots in top right of the window ... to reveal the More menu
  4. Select Map Network Drive from the menu
  5. This will bring up a window called Map Network Drive.  From here you can enter in the applicable information for the drive you intend to map
    1. First, specify a drive letter by clicking the drop down menu for Drive.
    2. Next, specify the specific folder you want to map.  For example: \\corpfs.nd.edu\sharedfoldername\
    3. Check the box for Reconnect at sign-in
    4. Uncheck the box for Connect using different credentials.  In most cases, this will force Windows to connect to the specified drive using your NetID/password.  If you need to connect to the drive using an alternate user name/password, you can check this box.
    5. Click Finish.
  6. Your new drive will now be mapped and mounted in the left pane of File Explorer.
