Obtaining the Cisco Secure Client (new installation)

The VPN (Virtual Private Network) is a network access service that permits remote systems to obtain a campus IP address and access the ND network as if they were locally connected. Certain restricted systems require a campus IP address for access.

Cisco Secure Client is the recommended method to access the VPN. To install Cisco Secure Client:

  • For Windows or Mac OSX machines the software can be downloaded from OIT Software Downloads
    • For university-owned and managed systems it can be installed from Software Center or Self-Service
  • For Android devices, the software can be downloaded from the Google Play store here
  • For iOS devices, the software can be downloaded from the App Store here

Note: Chrome OS, Windows RT and Windows Mobile do not support the VPN implementation used by Notre Dame at this time, which will limit the ability to access ND services from remote locations while using unsupported platforms. See Google's support page for additional information.

Connecting to the VPN (Cisco Secure Client)

After installing Cisco Secure Client, you can connect to the VPN by launching the program and entering the appropriate server address.

Mac: Go to Finder → Applications → Cisco → Cisco Secure Client

Windows: Go to Start → Cisco → Cisco Secure Client

  • If you are faculty or staff, use the address: vpnaccess.nd.edu
  • If you are a student, use the address: vpnaccess.nd.edu/go

Then select Connect and respond to the login prompts. Click Accept when prompted to complete the connection.

Using a Secure Key (Yubikey) for Two-Step Authentication

The VPN client uses its own browser for authentication and may not work with a Yubikey, particularly on Macintosh computers. If you are experiencing issues while using YubiKey to connect to the VPN, upgrade to Cisco Secure Client version 5.0.01242 or newer.
