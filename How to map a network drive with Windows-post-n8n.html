Question

How do I map a network drive in Windows?

Back to top

Answer

For Windows 10

  1. Open File Explorer from the Start Menu (Start > Windows System > File Explorer) or by pressing the Windows key + E. File Explorer may also be pinned to the taskbar and appears as a manila folder icon.

    ALT TEXT SUGGESTION: File Explorer icon highlighted on the Windows 10 taskbar.

  2. On the left-hand side, click This PC (or Windows).
  3. Click the Computer tab in the top menu bar.
  4. Click Map network drive from the toolbar that appears below.
  5. This will open the Map Network Drive window. Fill in the following:
    1. Select a drive letter from the Drive dropdown menu.
    2. Enter the folder path to map. Example: \corpfs.nd.edu\sharedfoldername
    3. Check Reconnect at sign-in.
    4. Uncheck Connect using different credentials (unless you need an alternate username/password).
    5. Click Finish.
  6. The drive will be mapped and will appear in the File Explorer left pane.

For Windows 11

  1. Open File Explorer from the Start Menu or press the Windows key + E. It may also be pinned to your taskbar as a manila folder icon.

    2. *** IMAGE PLACEHOLDER — ACTION NEEDED ***

    INTENDED IMAGE: file-explorer-icon-windows11.png

    How to insert images in the Knowledge Base

  2. Select This PC from the left pane.
  3. Click the three dots (...) in the top-right corner to reveal the More menu.
  4. Select Map network drive from the list.
  5. In the Map Network Drive window, configure the following:
    1. Select a drive letter from the Drive dropdown menu.
    2. Enter the folder path to map. Example: \corpfs.nd.edu\sharedfoldername
    3. Check Reconnect at sign-in.
    4. Uncheck Connect using different credentials (or check it if using alternate credentials).
    5. Click Finish.
  6. Your mapped drive will appear in the File Explorer left pane.
