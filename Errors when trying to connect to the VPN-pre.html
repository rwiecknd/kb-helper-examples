On 6/22/21 the VPN service was updated to require that the device connecting to it must be using version 4.10 or later of the Cisco AnyConnect Client. Most Windows and Macintosh computers will automatically install the latest version when attempting to connect to vpnaccess.nd.edu.  

If your computer does not automatically update, please visit OIT Software Downloads, locate the link for your operating system, download and install the new version.

Check your VPN version

You can check what version you have installed by:

  1. Launch the VPN client
  2. Click on the information button in the bottom left and locate the version number in the window that appears

  3. If you have version 4.6 or later when you attempt to connect to vpnaccess.nd.edu the server will attempt to automatically upgrade your system to the current supported version.  If that is unsuccessful, see further information below.

 

Errors which will require manual update from OIT Software Downloads are shown below:

Windows with a OneDrive error

Some installations of AnyConnect will provide a OneDrive error that states "No access permissions to the item." In these cases, you will need to perform the following steps to resolve the issue:

  1.  Download the latest version from the OIT Software Downloads page
  2.  Run the installer, and when given the option to "Repair, Modify, or Remove" choose Remove
  3.  Finish uninstalling the software
  4.  Restart the computer
  5.  Run the installer again. This time it should install the software properly
