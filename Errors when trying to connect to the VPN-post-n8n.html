Check your VPN version

On 6/22/21 the VPN service was updated to require that the device connecting to it must be using version 4.10 or later of the Cisco AnyConnect Client. Most Windows and Macintosh computers will automatically install the latest version when attempting to connect to vpnaccess.nd.edu.

If your computer does not automatically update, please visit OIT Software Downloads, locate the link for your operating system, and install the updated version.

You can check what version you have installed by:

  • Launching the VPN client
  • Clicking on the information button in the bottom left and locating the version number in the window that appears

Windows with a OneDrive error

Some installations of AnyConnect will provide a OneDrive error that states "No access permissions to the item." In these cases, perform the following steps:

  • Download the latest version from the OIT Software Downloads page.
  • Run the installer. When prompted with "Repair, Modify, or Remove", choose Remove.
  • Finish uninstalling the software.
  • Restart the computer.
  • Run the installer again. This time it should install the software properly.

Linux & Unsupported Operating Systems

Note: Computers with old operating systems (Windows 7 or earlier, or Mac OS prior to 10.14) may not be able to install or run the new VPN client. You will need to update your operating system and/or hardware first.

For support, contact your departmental IT group or, for personal devices, your preferred computer service provider.
